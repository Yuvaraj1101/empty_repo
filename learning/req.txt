Difference between venv and Conda

venv:
- Creates an isolated Python environment.
- Uses pip to install packages.
- Uses the existing Python installation.
- Best for Python development.

Conda:
- Creates an isolated environment.
- Manages Python versions and packages.
- Uses conda to install packages.
- Best for Data Science, Machine Learning, and AI projects.
